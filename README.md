# Background

A html5 puzzle game implemented with [lufylegend](https://github.com/lufylegend/lufylegend.js).

# Install

```bash
git clone https://github.com/ChenYi015/html5-puzzle.git
```

Open the ` puzzle.html` with your web browser.
